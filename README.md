# App.Bin.CodeEditor

应用包名：code.editor

显示名称：代码编辑器

版本：1.0.0

发布者：Flex_7746

## 应用说明

VS Code 同源库，支持多种编码文件的操作，更有语法高亮、代码补全。

## 本地运行

```bash
npm run install
npm run dev:frontend

# 另起 bash，xxxx 替换成访问密钥
USER_PSWD=xxxx npm run dev:backend
```

## 本地构建

> 请提前安装好 fnpack，如果不希望打包 fpk，可以执行 build:app

```bash
npm run install
npm run build
```
